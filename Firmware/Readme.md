# Chibios on STM32F103

## IDE

### GNU ARM Eclipse
Install Eclipse in CDT Edition
*	go to the Eclipse menu → Help → Eclipse Marketplace…
*   in the Find field, enter GNU ARM Eclipse
*   click the Go button

[Manual](https://gnuarmeclipse.github.io/plugins/install/)
